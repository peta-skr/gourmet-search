## アプリケーション機能

### 機能一覧

- 飲食店検索：ホットペッパーグルメサーチ API を使用して、現在地からの距離を選択して飲食店を検索する。
- 飲食店情報取得：ホットペッパーグルメサーチ API を使用して、飲食店の詳細情報を取得する。

### 画面一覧

- 検索画面：距離（半径）を指定して飲食店を検索する。
- 一覧画面：検索結果の飲食店を一覧表示する。
- 詳細画面：一つの飲食店の詳細情報を表示する。
